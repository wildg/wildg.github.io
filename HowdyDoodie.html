Work:

Cybera & Callysto
April 2019
Built Jupyter Noteboooks to teach academic material to students ranging from middle school to university. These notebooks were coded using Python and HTML, with an emphasis on interactive and visual learning. The project was completed in association with the University of Calgary and the Pacific Institute for the Math Sciences (PIMS) as well as CanCode to create a greater sense of interest in the sciences. It utilized a SCRUM development cycle with weekly sprints on Jira that led into demos and feedback, culminating in a public GitHub repository containing dozens of fascinating notebooks.
